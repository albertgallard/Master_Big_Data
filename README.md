# Master_Big_Data
Proyecto que conecta herramientas de Big Data con data en tiempo real para consumir y plasmar su información masiva en tiempo real

Los componentes utilizados son:


<ul>
<li>HDFS</li>
<li>MapReduce</li>
<li>Elastic Search</li>
<li>Kibana</li>
<li>Twitter Api</li>
<li>EMT Api</li>
</ul>
